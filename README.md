# 医用气动辅助手套项目 (Medical Pneumatic Assistive Glove)

> 基于 **正点原子领航者 ZYNQ7020** + **7" 800×480 触摸 LCD** 的纯 **PL/PS** 端实现
> 创建日期：2026-05-11

---

## 项目简介

本项目旨在设计并实现一款医用气动辅助手套，目标用户是脑卒中后手指屈曲挛缩、神经损伤的患者，提供 **被动辅助抓握 / 松手** 训练。

### 核心功能
- **`+`** 按钮：触摸后气泵加压，软体手指弯曲，实现 **抓握**
- **`-`** 按钮：触摸后泄气阀打开，软体手指回弹，实现 **松手**
- LCD 触摸屏实时显示压力数值、档位、状态
- 全部控制逻辑在 ZYNQ7020 的 **PL + PS** 端实现，不依赖外部 MCU

### 工作原理一句话
```
触摸 LCD → ZYNQ7020 → 驱动板 → 微型气泵/电磁阀 → 气体进入软体手指 → 弯曲抓握
                                                       │
                                                       └ 气压传感器 → 反馈回 ZYNQ → 闭环控制
```

```
[触摸 LCD] ◄──RGB+IIC──► [ZYNQ7020 PL+PS]
                              │
                              │AXI-GPIO
                              ▼
                         [ULN2803 驱动板]
                              │
                  ┌───────────┼────────────┐
                  ▼           ▼            ▼
              [气泵]      [充气阀 V1]   [泄气阀 V2]
                  │           │            │
                  └───────────┼────────────┘
                              ▼
                     [软体气动手指 ×5]
                              │
                       [气压传感器] ──► IIC ──► ZYNQ7020
```

## 关键技术点

1. **PL 端 LCD 控制**：参考正点原子《领航者 ZYNQ 之 FPGA 开发指南》第 20 章
2. **GT9147 触摸驱动**：IIC 时序、5 点触摸、坐标解析
3. **AXI4-Lite 寄存器**：PS 配置 PL 中的 PWM / 阀门状态
4. **IIC 主机模块**：读取 XGZP6857D 数字气压传感器
5. **安全联锁**：硬件比较器 + 看门狗 + 失电常开泄气阀
6. **状态机**：IDLE → GRIP → HOLD → RELEASE → IDLE

## ⚠️ 安全声明（重要）

> 本项目为 **个人学习实验性** 项目，**不是医疗器械**，**不可替代** 临床医疗用品。

- 工作压力严格限制 **≤ 80 kPa**（硬件 + 软件双保险）
- 必须在 **没人佩戴** 的情况下完成所有调试
- 第一次穿戴时建议用 **假肢 / 健康人手** 测试 1 周再考虑实验对象
- 泄气阀必须选 **常开型 (NO)** —— 断电时自动放气
- 准备物理 **急停按钮**（红色蘑菇头），断电源可强制泄压
- 任何人佩戴前请获取 **书面知情同意**，并在 **专业人员监督** 下进行
